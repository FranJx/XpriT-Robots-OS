## Notas

- **Placa Xmotion**: Controladora principal con soporte para Arduino Leonardo
- **Motores**: Motor DC 6V 750RPM para propulsión diferencial
- **Sensores JS40F**: Sensores IR para detección de oponentes (rango ~40cm)
- **Ruedas JS2622**: Par de ruedas de aluminio con silicona para tracción
- **Sensores ML1**: Sensores de línea para detección de bordes/acantilado
- **Batería**: LiPo 3S 350mAh 11.1V para alimentación del robot
